# meteor-alipay-direct-pay-example

## Installation

```shell
git clone git@github.com:yongjun925/meteor-alipay-direct-pay-example.git
cd meteor-alipay-direct-pay-example
meteor
```

## Usage
```
$ meteor add yongjun:meteor-alipay-direct-pay-example
```
